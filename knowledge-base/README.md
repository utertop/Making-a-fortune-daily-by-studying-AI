# AI Signal Radar Knowledge Base

This directory is the primary Markdown knowledge base for v0.1.

Current structure:

- `daily/`: daily AI radar notes
- `weekly/`: weekly trend summaries
- `projects/`: GitHub project analysis
- `concepts/`: concepts and engineering patterns
- `people/`: key people and viewpoints
- `companies/`: company and lab updates
- `comparisons/`: cross-project and concept comparisons
- `templates/`: reusable Markdown templates for generated drafts
